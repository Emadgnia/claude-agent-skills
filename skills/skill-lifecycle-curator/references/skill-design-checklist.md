# Skill Design Checklist

- [ ] Name is hyphen-case and under 64 chars
- [ ] Frontmatter has only `name` and `description`
- [ ] Description states task + trigger conditions
- [ ] Body is concise and operational
- [ ] References are linked directly from `SKILL.md`
- [ ] Scripts are tested on representative input
- [ ] `agents/openai.yaml` reflects current skill purpose
- [ ] Overlap with existing skills is minimized
