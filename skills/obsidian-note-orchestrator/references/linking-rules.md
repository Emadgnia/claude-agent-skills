# Linking Rules

Use these conventions for reliable Obsidian navigation.

- Use title-cased note names with stable date tokens.
- Keep one canonical name per note; avoid near-duplicates.
- Link weekly note from each daily note and vice versa.
- Update index notes when moving or renaming folders.
- Keep link labels human-readable; avoid raw file extensions in links.

Checklist after restructure:

- [ ] Daily note links resolve
- [ ] Weekly note links resolve
- [ ] Index/home links resolve
- [ ] No orphaned previous-week notes
